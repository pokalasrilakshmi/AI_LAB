# PyScript Demo
This is a simple example showing how **Python code can run inside an HTML page** using **PyScript**.

## What this demo does
- Displays the text "hello" using Python
- Prints numbers between 1 and 20 by incrementing by 2 using a Python loop
- Uses basic HTML and CSS for layout

## Technologies used
- HTML
- CSS
- PyScript (Python in the browser)
  
## Output
- A box showing the word `hello`
- Another box showing odd numbers (1, 3, 5, ..., 19)
